# Taiwan Transcriptions

There are two Taiwan based transcription systems:

- [POJ](https://en.wikipedia.org/wiki/Pe%CC%8Dh-%C5%8De-j%C4%AB)
- [TL](https://en.wikipedia.org/wiki/Official_romanisation_scheme_for_Taiwanese_Hokkien)

As such we probably need variants for them. Here I propose:

- nan-Latn-TW for TL
- nan-Latn-TW-x-transpoj for POJ

since TL is the default latn transcription in Taiwan.

Not sure how nan-Latn-CN is written. In fact it looks like it's only written in Latn in TW so change the default there.
